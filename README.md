# Bahasa-lo REPL

Ultimate REPL untuk Bahasa Lo, dibuat di Python. REPL ini mendukung:

- Sintaks sederhana: `tulis`, `jika`, `apabila`, `Maka`
- File manager Linux vibes (`ls`, `cat`)
- Plugin system Python (`plugin`, `plugin -m`)
- Download file & repositori ke folder `downloads/` (`wget`, `curl`, `git`)
- Pindah file dari `downloads/` ke folder lain (`pindah <file> <folder>`)
- Proot-distro smooth (install/login distro)
- Backup otomatis session & data (`simpan`)
- Root mode prompt `[Root]>` (`root -a`)
- Menu bantuan lengkap (`bantuan`)
- Admin menu (`admin`) untuk tweak repo & update paket
