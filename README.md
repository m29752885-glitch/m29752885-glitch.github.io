# Bahasa Lo - Mini Interpreter REPL

Ini adalah bahasa pemrograman super simpel versi lo sendiri.

## Fitur
- `(+) >` terminal interaktif ala Python
- `tulis "..."` → print ke layar
- Alias `Echo "..."` → sama dengan `tulis`
- Variabel sederhana & ekspresi matematika
- Keyword alias:
  - `jika` → `if`
  - `apabila` → `elif`
  - `Maka` → `:`
- Multi-line block support
- `apt install paket` dijalankan via system dengan progress bar (tanpa sudo)
- Baris komentar dengan `#`
